Чтобы отправить новые изменения в коде на GitHub, вам нужно выполнить стандартную цепочку из трех команд в терминале.

Добавьте измененные файлы в индекс GitЭта команда находит все новые и измененные файлы в проекте и готовит их к сохранению:
git add .

Зафиксируйте изменения (создайте коммит)Обязательно напишите в кавычках короткое пояснение, что именно вы изменили в коде (например, "Добавил обработку исключений" или "Исправил баг в TcpClient"):
git commit -m "Опишите ваши изменения здесь"

Отправьте изменения на GitHubЗагрузите ваш коммит в удаленный репозиторий:
git push
